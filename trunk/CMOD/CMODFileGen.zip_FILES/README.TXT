========================
CMODFileGenerator
========================

- Extract the contents of CMODFileGen.zip to a directory somewhere.

- In Windows, simply double click on CMODFileGen.jar to execute it as a java program.
     (Assuming you have a JRE installed on your computer)

- In Linux, open a command prompt and change the permissions on CMODFileGen.jar to allow executing the file.
     (chmod +x CMODFileGen.jar)
  Then, execute the program by typing "CMODFileGen.jar".
